# TitleSlide

## What It Is
A complete title slide layout that combines a TitleBar header with a centered body content area below. A convenience layout that wraps the common pattern of header plus centered content for title slides.

## Visual Appearance
The layout displays a TitleBar header at the top with title, optional subtitle, optional icon, and optional right section content. Below the header, a large centered content area provides space for additional content such as hero text, images, or call-to-action buttons. The header uses a bottom border for separation, and the content area is centered both horizontally and vertically within the remaining space.

## When to Use
- For title slides and opening slides
- When you need a standard header plus centered content pattern
- In presentations that start with title slides
- For section divider slides with titles
- When consistent title slide formatting is needed
- As a quick way to combine TitleBar with centered content

## Configuration Options
- **Title**: Main heading text for the header (required)
- **Subtitle**: Optional secondary text in the header
- **Right section**: Optional custom content in the header's right side
- **Children**: Content displayed in the centered body area
- **Header size**: Large header sizing for prominence
- **Bordered header**: Bottom border separator enabled
- **Entrance animation**: Optional animation effects when the slide appears

## Works Well With
- TitleBar component (used in header)
- SlideTitle for additional headings in content area
- SlideText for body content
- CTAButton for call-to-action buttons
- HeroSpotlight for hero content
- SlideImage for title slide visuals
