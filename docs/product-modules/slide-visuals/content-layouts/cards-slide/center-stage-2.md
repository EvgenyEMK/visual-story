# Center Stage (2 Items)

## Overview
A centred layout displaying two content cards side by side in the middle of the slide. Like Center Stage but for a pair of items that share the spotlight equally. The surrounding whitespace frames the two cards, drawing attention to the pair as a cohesive unit. Can be used with or without a slide header.

## Visual Structure
```
┌─────────────────────────────────────┐
│                                     │
│       ┌──────────┬──────────┐       │
│       │          │          │       │
│       │  CARD 1  │  CARD 2  │       │
│       │          │          │       │
│       └──────────┴──────────┘       │
│                                     │
└─────────────────────────────────────┘
```

## When to Use
- Presenting two key metrics or KPIs side by side
- Before/after comparisons in a focused, centred layout
- Two complementary products, features, or options
- Dual quotes, testimonials, or highlights
- Any slide where exactly two items should share centre-stage focus

## Content Recommendations
- **Card 1 & Card 2** (each):
  - Metric card with value, label, and optional delta
  - Feature card with icon, title, and short description
  - Product image with caption
  - Quote or testimonial with attribution
  - Stat or data point with visualisation
  - Before/after comparison item
