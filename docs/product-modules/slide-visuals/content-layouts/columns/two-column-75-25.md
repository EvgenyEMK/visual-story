# Two Columns (75/25)

## Overview
An asymmetric two-column layout with a wide left pane taking 75% of the width and a narrow right pane occupying 25%. This structure emphasizes the primary content on the left while reserving space for supplementary notes, metadata, quick stats, or supporting information that enhances but doesn't compete with the main message. Can be used with or without a slide header.

## Visual Structure
```
┌────────────────────────────┬────────┐
│                            │        │
│       LEFT COLUMN          │ RIGHT  │
│          (75%)             │ (25%)  │
│                            │        │
│                            │        │
│                            │        │
└────────────────────────────┴────────┘
```

## When to Use
- Slides where main content needs maximum space with supporting information visible
- Presenting detailed content alongside quick stats or key metrics
- Showing primary narrative with supplementary notes or annotations
- Content where metadata, tags, or labels enhance understanding without dominating
- Slides requiring a "notes panel" or "insights sidebar" alongside main content

## Content Recommendations
- **Left Column (75%)**:
  - Main narrative content, descriptions, or explanations
  - Primary charts, graphs, or data visualizations
  - Detailed feature lists or specifications
  - Long-form text or article-style content
  - Images or media that need prominent display
- **Right Column (25%)**:
  - Key statistics or metrics
  - Supplementary notes or annotations
  - Tags, labels, or metadata
  - Quick reference information
  - Callout boxes or highlight boxes
  - Related links or next steps
