# Blank Canvas

## Overview
An empty slide with no predefined regions or layout constraints. Items are placed using absolute positioning, giving complete creative freedom to arrange content anywhere on the slide. This layout is the ultimate flexibility option, allowing for unique, non-standard arrangements that don't fit into traditional grid or column structures. Perfect for highly visual slides, custom compositions, or layouts that break conventional presentation patterns. Can be used with or without a slide header.

## When to Use
- Highly visual slides with custom compositions
- Non-standard layouts that don't fit template patterns
- Slides requiring precise positioning of multiple elements
- Artistic or creative presentations with unique arrangements
- Overlapping elements or layered content designs
- Slides where content placement is part of the visual message
- Prototyping custom layout ideas before creating templates

## Content Recommendations
- **Any content type** can be placed anywhere:
  - Images with custom positioning and sizing
  - Text blocks positioned for visual flow
  - Overlapping elements or layered designs
  - Custom shapes, lines, or visual elements
  - Annotations or callouts positioned precisely
  - Mixed media arranged in unique compositions
  - Background elements with foreground content
