# Content

## Overview
The most versatile content layout — a single full-width content area that fills the entire slide (or the area below the header, if present). This layout provides maximum flexibility for content presentation, accommodating any visual arrangement from text blocks and charts to images, grids, or interactive components. Can be used with or without a slide header.

## Visual Structure
```
┌─────────────────────────────────────┐
│                                     │
│         CONTENT AREA                │
│      (Full width, flexible)         │
│                                     │
│   Can contain: text, charts,        │
│   images, grids, components         │
│                                     │
└─────────────────────────────────────┘
```

## When to Use
- General-purpose slides that need flexible content space
- Content-heavy slides with mixed media types
- Slides where the content doesn't naturally split into columns or grids
- Presentations requiring maximum content area without layout constraints
- Any slide where you need a single, unstructured content region

## Content Recommendations
- **Content Area**:
  - Long-form text blocks with headings and paragraphs
  - Data visualizations (charts, graphs, tables)
  - Images or image galleries
  - Feature lists or bullet points
  - Interactive components or embedded media
  - Mixed content layouts using nested containers
