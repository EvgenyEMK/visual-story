# Script Input

> **Parent module:** [Slide Editor](../README.md)
> **Status:** `planned`

## Overview

The script input feature allows users to enter narrative content that will be transformed into animated slides. It serves as the primary entry point for content creation and includes section markers for slide segmentation.

## Features

| Feature | File | Summary |
|---------|------|---------|
| Script Input | [script-input.md](./script-input.md) | Text editor with section markers, auto-save, character count, and intent/tone selection |
