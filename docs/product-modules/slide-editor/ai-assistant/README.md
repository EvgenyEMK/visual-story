# AI Assistant

> **Parent module:** [Slide Editor](../README.md)
> **Status:** `planned`

## Overview

The AI Assistant provides intelligent automation within the slide editor — suggesting visual elements, providing script feedback, and generating assets. In MVP, this focuses on icon suggestions from a curated library and script structure feedback via OpenAI.

## Features

| Feature | File | Summary |
|---------|------|---------|
| Visual Suggestions | [visual-suggestions.md](./visual-suggestions.md) | AI-powered icon and visual element recommendations based on slide content |
| Script Feedback | [script-feedback.md](./script-feedback.md) | OpenAI-based suggestions for improving script structure, hook, and call-to-action |
| Asset Generation | [asset-generation.md](./asset-generation.md) | AI image and asset generation for slides (Phase 2 — stock library placeholders in MVP) |
