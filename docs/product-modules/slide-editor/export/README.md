# Export

> **Parent module:** [Slide Editor](../README.md)
> **Status:** `planned`

## Overview

Export features allow users to produce shareable outputs from their presentations. This includes rendering to MP4 video (via Remotion Lambda) and generating embeddable/shareable links. The web player experience itself is documented separately under [slide-player/](../../slide-player/).

## Features

| Feature | File | Summary |
|---------|------|---------|
| Video Export | [video-export.md](./video-export.md) | Render presentation as MP4 (1080p) via Remotion Lambda with synchronized audio |
| Embed & Sharing | [embed-sharing.md](./embed-sharing.md) | Generate embed codes and shareable URLs for web distribution |
