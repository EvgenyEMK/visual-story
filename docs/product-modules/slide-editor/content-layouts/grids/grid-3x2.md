# Grid 3x2

## Overview
A layout featuring six equal cells arranged in a 3x2 grid (3 columns, 2 rows). This creates a compact but organized presentation structure. Perfect for showcasing six related items — features, icons, team profiles, or step-by-step processes — where each item receives equal visual weight. Can be used with or without a slide header.

## Visual Structure
```
┌────────────┬────────────┬───────────┐
│            │            │           │
│  TOP LEFT  │ TOP MIDDLE │ TOP RIGHT │
│            │            │           │
├────────────┼────────────┼───────────┤
│            │            │           │
│  BOT LEFT  │ BOT MIDDLE │ BOT RIGHT │
│            │            │           │
└────────────┴────────────┴───────────┘
```

## When to Use
- Showcasing six features or capabilities
- Presenting six team members or stakeholders
- Displaying six product options or variations
- Showing six steps in a process or workflow
- Organizing six data points or metrics
- Presenting six icons or visual elements
- Any content that naturally divides into six equal parts

## Content Recommendations
- **Each Grid Cell** can contain:
  - Feature icons with labels and brief descriptions
  - Team member photos with names and titles
  - Product thumbnails with captions
  - Process step indicators with step numbers and descriptions
  - Metric cards with values and labels
  - Category badges with representative visuals
  - Comparison items with key attributes
  - Visual elements that work well in compact spaces
