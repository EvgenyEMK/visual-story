# Three Columns

## Overview
A balanced layout featuring three equal-width vertical panes, creating a three-way split (33% / 33% / 33%). This structure is ideal for comparing three options, presenting three pillars or principles, or organizing content into three distinct categories. The equal division ensures all three columns receive equal visual weight. Can be used with or without a slide header.

## Visual Structure
```
┌────────────┬────────────┬───────────┐
│            │            │           │
│   LEFT     │  MIDDLE    │   RIGHT   │
│  COLUMN    │  COLUMN    │  COLUMN   │
│   (33%)    │   (33%)    │   (33%)   │
│            │            │           │
│            │            │           │
└────────────┴────────────┴───────────┘
```

## When to Use
- Comparing three options, products, or solutions
- Presenting three pillars, principles, or core concepts
- Organizing content into three categories or groups
- Showing a three-step process or timeline
- Displaying three related metrics or data points side-by-side
- Feature showcases with three main highlights

## Content Recommendations
- **Left Column (33%)**:
  - First option, concept, or category
  - First pillar or principle
  - Step 1 or beginning of process
  - First metric or data point
- **Middle Column (33%)**:
  - Second option, concept, or category
  - Second pillar or principle
  - Step 2 or middle of process
  - Second metric or data point
- **Right Column (33%)**:
  - Third option, concept, or category
  - Third pillar or principle
  - Step 3 or end of process
  - Third metric or data point
