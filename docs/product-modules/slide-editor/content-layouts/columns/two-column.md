# Two Columns (50/50)

## Overview
A balanced layout featuring two equal-width vertical panes side-by-side, creating a 50/50 split. This layout excels at presenting parallel content, comparisons, or complementary information side-by-side. The equal division ensures both columns receive equal visual weight. Can be used with or without a slide header.

## Visual Structure
```
┌──────────────────┬──────────────────┐
│                  │                  │
│   LEFT COLUMN    │   RIGHT COLUMN   │
│     (50%)        │     (50%)        │
│                  │                  │
│                  │                  │
│                  │                  │
└──────────────────┴──────────────────┘
```

## When to Use
- Comparing two options, concepts, or states (before/after, pros/cons)
- Presenting text alongside visuals (left: description, right: image/chart)
- Showing two parallel content blocks that relate to each other
- Split-screen presentations where both sides need equal emphasis
- Side-by-side feature comparisons or product specifications
- Without header: title can be integrated into the left pane as part of the narrative

## Content Recommendations
- **Left Column (50%)**:
  - Text descriptions, bullet lists, or narrative content
  - "Before" state, pros, or first option details
  - Step-by-step instructions or process descriptions
  - Slide title and subtitle (when used without header)
  - Feature lists or specifications
- **Right Column (50%)**:
  - Visual content (images, charts, diagrams)
  - "After" state, cons, or second option details
  - Results, outcomes, or supporting data
  - Data visualizations or interactive components
  - Product images or hero visuals
