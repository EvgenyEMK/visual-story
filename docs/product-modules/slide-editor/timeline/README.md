# Timeline

> **Parent module:** [Slide Editor](../README.md)
> **Status:** `planned`

## Overview

The timeline provides a horizontal overview of all slides and their relative timing. Users can navigate between slides, reorder them via drag-and-drop, and understand the overall presentation flow.

## Features

| Feature | File | Summary |
|---------|------|---------|
| Timeline View | [timeline-view.md](./timeline-view.md) | Horizontal slide strip with thumbnails, duration indicators, drag-to-reorder, and playback controls |
