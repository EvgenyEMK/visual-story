# Four Columns

## Overview
A balanced layout featuring four equal-width vertical panes, creating a four-way split (25% / 25% / 25% / 25%). This structure is ideal for comparing four options, presenting quarterly data, or organising content into four distinct categories. The equal division ensures all four columns receive equal visual weight. Can be used with or without a slide header.

## Visual Structure
```
┌─────────┬─────────┬─────────┬─────────┐
│         │         │         │         │
│  COL 1  │  COL 2  │  COL 3  │  COL 4  │
│  (25%)  │  (25%)  │  (25%)  │  (25%)  │
│         │         │         │         │
│         │         │         │         │
│         │         │         │         │
└─────────┴─────────┴─────────┴─────────┘
```

## When to Use
- Comparing four options, products, or solutions
- Presenting quarterly data (Q1, Q2, Q3, Q4)
- Organising content into four categories or groups
- Showing a four-step process or timeline
- Displaying four related metrics or data points side-by-side
- Feature showcases with four main highlights
- Pricing tiers or plan comparisons

## Content Recommendations
- **Column 1 (25%)**:
  - First option, concept, or category
  - Q1 data or first phase
  - Step 1 or beginning of process
  - First metric or data point
- **Column 2 (25%)**:
  - Second option, concept, or category
  - Q2 data or second phase
  - Step 2 of process
  - Second metric or data point
- **Column 3 (25%)**:
  - Third option, concept, or category
  - Q3 data or third phase
  - Step 3 of process
  - Third metric or data point
- **Column 4 (25%)**:
  - Fourth option, concept, or category
  - Q4 data or final phase
  - Step 4 or end of process
  - Fourth metric or data point
