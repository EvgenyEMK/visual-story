# Center Stage (3 Items)

## Overview
A centred layout displaying three content cards in a row in the middle of the slide. Like Center Stage but for a trio of items that share the spotlight equally. The surrounding whitespace frames the three cards as a cohesive group, ideal for presenting three pillars, key stats, or step overviews. Can be used with or without a slide header.

## Visual Structure
```
┌─────────────────────────────────────┐
│                                     │
│    ┌─────────┬─────────┬─────────┐  │
│    │         │         │         │  │
│    │ CARD 1  │ CARD 2  │ CARD 3  │  │
│    │         │         │         │  │
│    └─────────┴─────────┴─────────┘  │
│                                     │
└─────────────────────────────────────┘
```

## When to Use
- Presenting three key metrics or KPIs
- Three pillars, principles, or core values
- Three featured products, plans, or options
- A three-step process overview
- Triple comparison of options or tiers
- Any slide where exactly three items should share centre-stage focus

## Content Recommendations
- **Card 1, Card 2 & Card 3** (each):
  - Metric card with value, label, and optional delta
  - Feature card with icon, title, and short description
  - Pricing tier or plan summary
  - Process step with number, title, and brief detail
  - Product image with caption
  - Pillar or value with icon and description
