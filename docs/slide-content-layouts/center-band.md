# Center Band

## Overview
A full-width horizontal content bar centred vertically on the slide. Unlike Center Stage (which constrains content to a centred box), Center Band stretches edge to edge while keeping the content strip vertically centred with generous whitespace above and below. This creates a strong, cinematic feel that works especially well for section dividers and horizontal process flows. Can be used with or without a slide header.

## Visual Structure
```
┌─────────────────────────────────────┐
│                                     │
│                                     │
├─────────────────────────────────────┤
│          CENTER BAND                │
│      (full-width, centred)          │
├─────────────────────────────────────┤
│                                     │
│                                     │
└─────────────────────────────────────┘
```

## When to Use
- Section or chapter title slides (the band holds the section name, without a header bar)
- Process flow summaries showing workflow stages in a horizontal line
- Timeline or pipeline visualisations that benefit from full width
- Horizontal navigation bars or step indicators
- Quote or callout strips that should span the full slide width
- Any content that is naturally horizontal and benefits from edge-to-edge space

## Content Recommendations
- **Center Band**:
  - Section title with optional subtitle or section number
  - Horizontal process flow with connected step nodes
  - Workflow stages (e.g., Brief -> Research -> Design -> Build -> Launch)
  - Timeline strip with milestones
  - Horizontal list of logos, icons, or badges
  - Quote or key message in a full-width strip
  - Progress bar or phase indicator
