# Center Stage

## Overview
A minimalist layout where the content area serves as a single centered region, drawing maximum attention to one focal element. This layout creates a spotlight effect, ensuring the primary content commands full visual impact. Ideal for title/cover slides, hero statements, big metrics, impactful quotes, or any content that should dominate the visual space without competing elements. Can be used with or without a slide header.

## Visual Structure
```
┌─────────────────────────────────────┐
│                                     │
│         ┌───────────────┐           │
│         │               │           │
│         │ CENTER STAGE  │           │
│         │   CONTENT     │           │
│         │               │           │
│         └───────────────┘           │
│                                     │
└─────────────────────────────────────┘
```

## When to Use
- Title slides or cover pages for presentations
- Hero statements or key messages that need maximum impact
- Large, prominent metrics or KPIs that should dominate
- Impactful quotes or testimonials with attribution
- Product launch announcements or major reveals
- Call-to-action slides with singular focus
- With header: featured metric or diagram with context title above
- Without header: title and content merge into a single centered composition

## Content Recommendations
- **Center Stage Area**:
  - Presentation title with subtitle or tagline
  - Large, bold metrics or KPIs with visual emphasis
  - Hero quotes or testimonials with speaker attribution
  - Product names or brand announcements
  - Single, high-impact image or visual
  - Key message or value proposition
  - Call-to-action buttons or primary CTAs
  - Key diagrams, flowcharts, or process visualizations
  - Company logos or brand elements
