# Two Columns (25/75)

## Overview
An asymmetric two-column layout with a narrow left pane occupying 25% of the width and a wide right pane taking 75%. This structure prioritizes the main content area on the right while providing a dedicated space for navigation, categorization, or supplementary information that doesn't require extensive space. Can be used with or without a slide header.

## Visual Structure
```
┌────────┬────────────────────────────┐
│        │                            │
│  LEFT  │       RIGHT COLUMN         │
│ (25%)  │          (75%)             │
│        │                            │
│        │                            │
│        │                            │
└────────┴────────────────────────────┘
```

## When to Use
- Slides where a list, menu, or navigation needs to accompany detailed content
- Presenting a legend or key alongside a large visualization
- Showing categories or filters next to main content
- Slides with metadata, tags, or labels that need to be visible but not prominent
- Content where the left sidebar provides context for the right-side detail view

## Content Recommendations
- **Left Column (25%)**:
  - Navigation list or menu items
  - Category labels or filters
  - Legend for charts or diagrams
  - Table of contents or section index
  - Tags, labels, or metadata
  - Quick reference information
- **Right Column (75%)**:
  - Main content area with detailed information
  - Large charts, graphs, or data visualizations
  - Detailed descriptions or explanations
  - Images or media that need prominent display
  - Interactive components or embedded content
